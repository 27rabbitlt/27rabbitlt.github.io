# Basics

## 二极管

我们简单理解为，电流总是从 P 结流向 N 结。在单向导通的时候二极管稳定的有一个 0.7V 的压降（当然具体得看手册）。在反向击穿的时候可以近似充当一个稳压元件，因为当反向电压超过击穿电压的时候电流增长会非常快。

二极管当然也是有电阻的。但是电阻这个词在高中语境里很特殊，我们直接定义为电压除以电流。而对于一个非线性元件，电阻可能会随着电压电流而变化。此时我们可能会关心微分电阻。微分电阻就是伏安特性曲线上的切线斜率；而电阻则是原点连线的斜率。微分电阻刻画了在这一状态下，你想提升一单位的电压或者是电流有多难。

对于二极管来说，我们有一个肖特基公式，描述了二极管的伏安特性曲线：$I = I_S(e^(V/U_T) - 1)$

### 三极管

在这里我们首先辨析名词。中文中三极管指的是 Bipolar Junction Transistor，也就是 BJT。这是一种 Transistor，中文中叫晶体管。常见的我们还有 MOSFET，它也是一种晶体管，但和三极管的工作原理并不相同。

所以晶体管（Transistor）包括三极管（BJT）和 MOSFET。

然而因为中文中三极的本意无非是三个引脚，所以很多别的东西比如MOSFET也会被称为三极管。

以下三极管都指代的是 BJT。

三极管的三个极分别是 base 基，collect 集电，emit 发射。一般 b 写在左边，c 写在上面，e 写在下面。

三极管有三个工作区，分别是截止区，放大区，饱和区。

当发射结反偏的时候，也即 Ube < Uon 的时候，三极管处于截止区。
当发射结正偏的时候，有两种可能。如果 Uce < Ube, i.e. Ucb < 0，集电正偏，那就是饱和区；如果 Uce > Ube, i.e. Ucb > 0，集电反偏，那就是放大区。

我们一般是用它的放大区的特性。此时 ic = beta * ib。beta 就是它的放大倍率。
